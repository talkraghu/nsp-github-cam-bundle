# Vendored artifact-bundle-builder

The Go sources in this directory are copied from the **`build-cam-artifact-bundle-builder`** tree in the NSP `cam` workspace:

- `main.go`
- `artifact-meta.go`

Preserve Nokia copyright and license headers from the upstream files. When updating, replace these files from upstream and re-run `./scripts/repack.sh` to verify the bundle still builds.
